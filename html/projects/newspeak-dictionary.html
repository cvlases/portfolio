// build + art

Newspeak Dictionary

I wanted to find the words AI had hollowed out. What I found instead was that the literal meanings were still there — waiting for the right literary framing to call them back.

RoleSolo developer / artist
ToolsNext.js, Modal, Gemma, SAE steering
TimeSpring 2025
CourseAdvanced Creative Code

// hook

This project probes words with physical and ecological weight — words like soil, root, and bloom — to ask what large language models default to, what they forget, and what can be recovered through framing.

// the problem

AI prose often flattens words that should feel sensory, local, and embodied. The initial thesis was that AI had stripped them down. The more interesting discovery was that the meanings were still latent inside the model.

// the pivot

The project got better when the thesis changed. Instead of proving loss, it became an experiment in retrieval: how technical steering and literary framing could surface something richer.

// process

  • Started with GPT-2 / GemmaScope exploration.
  • Moved toward Gemma-2-2B with SAE feature steering.
  • Built a vintage-glitch visual language to match the feeling of excavation.
  • Used the interface itself as part of the argument, not just a wrapper around outputs.

// what i learned

The strongest version of this project is not anti-AI in a broad way. It is more precise than that: a study in how model behavior, interface framing, and literary context shape meaning together.
