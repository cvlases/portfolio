// build + art

Portraits of Algorithmic Attention

Saliency maps show you where a neural network is “looking.” This project asks what happens when that attention becomes portraiture instead of pure explanation.

RoleSolo developer / artist
ToolsComfyUI, custom saliency node
TypeCreative coding + XAI
StatusStarter page

// what it is

This project uses a custom-built explainable AI / saliency workflow in ComfyUI to visualize model attention as a form of portraiture. Instead of treating saliency as a technical diagnostic only, it reframes attention as image-making.

// the question

What does it look like when a model “looks” at a face, a body, or a landscape? And how does that change when you present the answer as art rather than just explanation?

// why it matters

The project sits in a sweet spot between creative coding and interpretability. It makes AI attention legible, but it also makes it emotionally and aesthetically legible.

// process

  • Built or adapted a custom saliency node inside ComfyUI.
  • Used that workflow to generate portraits of model attention.
  • Treated the resulting outputs not just as debug information, but as a visual medium in their own right.
